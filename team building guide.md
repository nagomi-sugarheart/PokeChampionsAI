# パーティ構築の基本ガイド（ポケモンチャンピオンズ・シングル）

本ドキュメントは、種族値・技・特性などの「事実データ」だけでは出てこない、**構築の考え方・定石**をまとめたものである。提案の際は、まず本ガイドで全体の方向性（構築タイプ・勝ち筋）を定めてから、pokemon.csv等の個別データで肉付けすること。

---

## 1. 構築の基本プロセス

### STEP1：仮想敵の設定
UsedRanking/current_single.json（使用率上位）とseason_regulation_changes.mdを参照し、「現環境で何が多いか」「新規解禁/復権で急増しそうなものは何か」を把握する。仮想敵は3〜5体程度に絞る。

### STEP2：軸（勝ち筋）の決定
パーティ全体が**何で勝つか**を最初に1つ決める。「強いポケモンを6体並べる」のではなく、§2の構築タイプから1つを選び、それを起点に役割を埋めていく。軸が無いまま個別の強さだけで集めると、噛み合わない・詰みやすいパーティになりやすい。

### STEP3：役割の穴埋め
軸が機能するために必要な役割を埋める。代表的な役割：
* **起点作成**：相手を直接崩さず、後続のために展開する（壁・追い風・設置技・いかく等）
* **受け**：物理/特殊それぞれの仮想敵を受けられる駒
* **遂行者（アタッカー）**：実際に勝ち筋を遂行する駒（積みエース、高火力アタッカー等）
* **サイクル要員**：交代を繰り返してダメージレースを有利に進める駒

### STEP4：保険・対策技の確認
* 後攻を強要された場合の保険（先制技、トリックルーム、削り技）はあるか
* 仮想敵の中で**一貫して止まらない**駒がいないか（全員受けられない/抜かれる相手がいないか確認）
* ステルスロック等の設置技への耐性（軽減・無効化できる駒）はあるか

### STEP5：最終チェック
* タイプ弱点が集中していないか（同じタイプの技で複数体が同時に崩されないか）
* 状態異常（特にまひ・どく）への対策がパーティ内にあるか
* 詳細はチェックリスト（§4）を参照

## 2. 構築タイプの分類

各タイプについて「勝ち筋」「典型的な特性・技・道具の組み合わせ」「弱点・崩され方」を示す。

### 天候パ（あめ/にほんばれ/すなあらし/ゆき）
* **勝ち筋**：天候始動の特性で素早さ2倍を得たアタッカーが高速で全抜きする、または天候専用の威力補正・回復補正で押し切る
* **典型構成**：天候始動特性（あめふらし／ひでり／すなおこし）持ちの起点役＋天候下で素早さが上がる特性（すいすい／ようりょくそ／すなかき／ゆきかき）を持つアタッカー
* **崩され方**：天候始動役が早期に倒されると機能停止する。複数の天候パが同時に存在すると後出しの天候に上書きされる

### トリックルームパ
* **勝ち筋**：素早さの遅い高火力アタッカーを「トリックルーム」で素早さ逆転させ、本来なら抜かれる相手より先に行動して押し切る
* **典型構成**：トリックルーム使用者（耐久寄り）＋低速高火力アタッカー複数（こだわり系道具は相性が悪い場合がある＝技固定で逃げ場が無くなるため注意）
* **崩され方**：トリックルームのターン数（5ターン）内に決着がつかないと素早さが正常に戻り失速する。先制技や「トリックルーム」自体への先読み交代に弱い

### 積み構築（剣舞・つるぎのまい系／瞑想・ようきだま系）
* **勝ち筋**：1体のエースが能力ランクを積み（剣舞・りゅうのまい・つるぎのまい・めいそう等）、全抜きする
* **典型構成**：積み技を通すための起点（壁・後攻保険・素早さ操作）＋積みエース本体（耐久面のサポートアイテム：きあいのタスキ等）
* **崩され方**：積む前に削られる、後出しの受けに止められる、急所や追加効果で積みが無駄になる

### 壁パ
* **勝ち筋**：リフレクター／ひかりのかべ／オーロラベールで被ダメージを抑えながらサイクルを回し、相手の消耗を誘う
* **典型構成**：壁役（耐久高め）＋壁の恩恵を受けるサイクル要員。「ひかりのねんど」で壁の持続を延長すると安定感が増す
* **崩され方**：壁を貫通する変化技（壊れた壁を無視する効果は基本的に無いが、急所は壁を無視する点に注意）、壁が切れたタイミングでの集中攻撃

### 受けループ
* **勝ち筋**：回復技・特性（さいせいりょく、ほろびのうた等は除く一般的な回復技構成）と状態異常（どく等）の組み合わせで、相手のPPや行動回数を枯らして詰ませる
* **典型構成**：高耐久＋回復技＋設置技（じわじわ削る）＋マジックガード等の「天候・設置技ダメージを受けない」特性が刺さりやすい
* **崩され方**：積み技で一体ずつ突破される、「まもる」貫通技や急所連発、こちらの回復量を上回る高火力アタッカー

### 対面構築（バランス型）
* **勝ち筋**：1つの突破口に依存せず、相手の選出・展開に応じて柔軟に対応する。役割破壊（役割対象の不利な特性や技を覚えさせて初手から有利を取る）の集合体
* **典型構成**：各タイプ・各仮想敵に対して最低1体は有利を取れる駒を揃える。型がバレにくく、後出しが効きやすい
* **崩され方**：器用貧乏になりやすく、明確な勝ち筋を持つ構築（積み・天候等）に対して受け切れない場合がある

## 3. 特性駆動のシナジー例

特性を「個別のダメージ補正係数」ではなく「構築の起点」として捉えるための代表例。

| シナジー | 内容 |
|---|---|
| すいすい × あめ | 雨下で素早さ2倍。「あめふらし」持ちの起点役と組み合わせて高速アタッカー化 |
| ようりょくそ × にほんばれ | 晴れ下で素早さ2倍。「ひでり」持ちと組み合わせる。やけど無効化の副次効果も活きる |
| すなかき／すなのちから × すなあらし | 砂嵐下で素早さ2倍（すなかき）、いわ/じめん/はがね技1.3倍（すなのちから）。「すなおこし」持ちと組み合わせる |
| ゆきかき × ゆき | 雪下で素早さ2倍。「ゆきふらし」等の天候始動役と組み合わせる |
| こだいかっせい／クォークチャージ × ブーストエナジー | 天候・フィールドが無くても道具で発動条件を満たせる（最も高い能力が1.3倍、素早さなら1.5倍） |
| いかく × 交代エース／引き隊 | 相手の物理アタッカーの攻撃を1段階下げてから後続に繋ぐ、起点作成の代表例 |
| マジックガード × 受けループ／設置技 | 天候・どく・設置技ダメージを無効化。砂嵐や毒撒き構築への耐性として機能 |
| わざわい系 × 高火力アタッカー | 相手の該当ステータスを0.75倍にすることで、味方の数値受けが崩れにくくなる／崩しやすくなる |
| フィールド系特性（グラスメイカー等） × パラドックス特性 | フィールドそのものの威力補正に加え、該当特性持ちのステータス補正条件も同時に満たせる |
| ノーガード × 必中技 | 命中不安の高火力技（じしん以外の低命中技等）を必中にしつつ、相手の回避率上昇を無視できる |

## 4. 構築チェックリスト（最終確認用）

提案を出す前に、以下を一通り確認すること。

- [ ] パーティの勝ち筋（軸）を一言で説明できるか
- [ ] 軸を機能させるための起点（壁・追い風・素早さ操作・いかく等）があるか
- [ ] 仮想敵（使用率上位）の中で、**誰も止められない駒**がいないか
- [ ] 後攻を強要された場合の保険（先制技・トリックルーム・耐久調整）があるか
- [ ] タイプ弱点が3体以上で重複していないか
- [ ] ステルスロック等の設置技への耐性・対策があるか
- [ ] 状態異常（特にまひ・どく・ねむり）への対策が1つ以上あるか
- [ ] 積み技がある場合、それを通すための起点・保険技があるか
- [ ] 天候パ／トリックルームパの場合、始動役が落ちた後の代替プランがあるか

---

## 運用ルール
ユーザーから「構築を考えて」と依頼された場合、まず§1のSTEP1〜2（仮想敵設定・軸決定）から会話を始め、いきなり個別ポケモンの提案に飛ばないこと。軸が決まった後で、pokemon.csv等のデータを使って役割に合うポケモンを当てはめていく。
